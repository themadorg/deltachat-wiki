# Madmail Technical Documentation

This section contains in-depth technical documentation about Madmail's architecture, implementation details, and module reference.

Designed as a single-binary, high-performance alternative to traditional mail stacks, Madmail integrates SMTP, IMAP, and various helper services (like TURN and Shadowsocks) into a cohesive system.
