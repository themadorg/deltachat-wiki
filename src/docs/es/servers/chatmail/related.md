# Desarrollos de la comunidad

El desarrollo activo tiene lugar en el repositorio de GitHub [chatmail/relay](https://github.com/chatmail/relay).

Puedes consultar la [etiqueta 'chatmail' en el foro support.delta.chat](https://support.delta.chat/tag/chatmail) y pedir que te añadan a un chat de soporte no público para depurar problemas.

Conocemos tres esfuerzos de implementación alternativa en curso:

- **[Mox](https://github.com/mjl-/mox)**: Un servidor de correo en Golang. Se está trabajando en el soporte de todas las características y ajustes de configuración necesarios para funcionar como un relay de chatmail.
- **[Madmail](https://github.com/omidz4t/madmail)**: Un fork experimental del [Servidor de Correo Maddy](https://maddy.email/) optimizado para despliegues de chatmail. Proporciona una solución de binario único.
- **[Chatmail Cookbook](https://github.com/feld/chatmail-cookbook)**: Un Chef Cookbook para implementar un servidor relay. Soporta FreeBSD y utiliza DNS-01 para la validación de certificados.
