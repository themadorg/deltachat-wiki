# Documentación técnica de Madmail

Esta sección contiene documentación técnica detallada sobre la arquitectura de Madmail, los detalles de implementación y la referencia de módulos.

Diseñado como un binario único y una alternativa de alto rendimiento a las pilas de correo tradicionales, Madmail integra SMTP, IMAP y varios servicios auxiliares (como TURN y Shadowsocks) en un sistema cohesivo.

- **[Tutoriales](/es/docs/servers/madmail/technical/tutorials/setting-up)**: Guías técnicas paso a paso.
