# رویدادهای هسته دلتا چت

این لیست جامعی از رویدادهای منتشر شده توسط موتور هسته دلتا چت است. بات‌ها باید این رویدادها را گوش دهند تا به فعالیت‌های مختلف مانند پیام‌های ورودی، تغییرات اتصال یا به‌روزرسانی‌های پیکربندی واکنش نشان دهند.

| نام رویداد | توضیحات |
| :--- | :--- |
| **DC_EVENT_ACCOUNTS_BACKGROUND_FETCH_DONE** | اطلاع می‌دهد که دریافت پس‌زمینه تکمیل شده (یا زمانش تمام شده). |
| **DC_EVENT_ACCOUNTS_CHANGED** | اطلاع می‌دهد که لیست حساب‌ها تغییر کرده (حسابی حذف یا اضافه شده یا ترتیب حساب‌ها تغییر کرده). این رویداد فقط توسط مدیر حساب منتشر می‌شود. |
| **DC_EVENT_ACCOUNTS_ITEM_CHANGED** | اطلاع می‌دهد که خاصیتی از حساب مانند `is_configured`، `displayname`، `selfavatar` یا `private_tag` تغییر کرده. |
| **DC_EVENT_CALL_ENDED** | تماس ورودی یا خروجی پایان یافت، یا ظرف ۱ دقیقه پذیرفته نشد. |
| **DC_EVENT_CHANNEL_OVERFLOW** | اطلاع می‌دهد که برخی رویدادها به دلیل سرریز کانال رویداد نادیده گرفته شده‌اند. |
| **DC_EVENT_CHAT_DELETED** | چت حذف شد. |
| **DC_EVENT_CHAT_EPHEMERAL_TIMER_MODIFIED** | تایمر ناپدید شونده چت تغییر کرد. |
| **DC_EVENT_CHAT_MODIFIED** | چت تغییر کرد (مثلاً عنوان، آواتار یا اعضا). |
| **DC_EVENT_CHATLIST_CHANGED** | اطلاع می‌دهد که مجموعه چت‌ها یا ترتیب چت‌ها در لیست چت تغییر کرده. |
| **DC_EVENT_CHATLIST_ITEM_CHANGED** | اطلاع می‌دهد که همه یا یک آیتم لیست چت تغییر کرده و نیاز به بازنمایی دارد. اگر `chat_id` برابر 0 باشد، همه چت‌های قابل مشاهده نیاز به بازنمایی دارند. |
| **DC_EVENT_CONFIG_SYNCED** | یک مقدار پیکربندی همگام‌سازی شده چند دستگاهی تغییر کرد. |
| **DC_EVENT_CONFIGURE_PROGRESS** | اطلاع از پیشرفت پیکربندی شروع شده توسط `dc_configure()`. |
| **DC_EVENT_CONNECTIVITY_CHANGED** | اتصال به سرور تغییر کرد (مثلاً در حال اتصال، آنلاین، آفلاین). |
| **DC_EVENT_CONTACTS_CHANGED** | مخاطب(ها) ساخته شد، تغییر نام داده شد، تأیید شد، مسدود یا حذف شد. |
| **DC_EVENT_DELETED_BLOB_FILE** | هنگامی منتشر می‌شود که یک فایل blob (تصویر/پیوست) با موفقیت حذف شد. |
| **DC_EVENT_ERROR** | کاربر کتابخانه باید خطایی را به کاربر نهایی گزارش دهد. |
| **DC_EVENT_ERROR_SELF_NOT_IN_GROUP** | یک عمل نمی‌تواند انجام شود زیرا نیاز به عضویت در گروه دارد. |
| **DC_EVENT_IMAP_CONNECTED** | هنگامی منتشر می‌شود که اتصال IMAP برقرار شده و ورود موفقیت‌آمیز بوده. |
| **DC_EVENT_IMAP_INBOX_IDLE** | هنگامی منتشر می‌شود که قبل از رفتن به حالت IDLE در پوشه صندوق ورودی. |
| **DC_EVENT_IMAP_MESSAGE_DELETED** | هنگامی منتشر می‌شود که پیامی با موفقیت در سرور IMAP به عنوان حذف شده علامت‌گذاری شد. |
| **DC_EVENT_IMAP_MESSAGE_MOVED** | هنگامی منتشر می‌شود که پیامی با موفقیت در IMAP منتقل شد. |
| **DC_EVENT_IMEX_FILE_WRITTEN** | یک فایل صادر شده (مثلاً در حین پشتیبان‌گیری/صادرات). |
| **DC_EVENT_IMEX_PROGRESS** | اطلاع از پیشرفت وارد/صادرکردن شروع شده توسط `dc_imex()`. |
| **DC_EVENT_INCOMING_CALL** | یک تماس ورودی دریافت شد. |
| **DC_EVENT_INCOMING_CALL_ACCEPTED** | گیرنده تماس، تماس ورودی را در این دستگاه یا دستگاه دیگری پذیرفت. |
| **DC_EVENT_INCOMING_MSG** | یک پیام تازه دریافت و در پایگاه داده ذخیره شده. |
| **DC_EVENT_INCOMING_MSG_BUNCH** | دانلود دسته‌ای از پیام‌ها تمام شد. |
| **DC_EVENT_INCOMING_REACTION** | واکنشی به پیام ارسالی خود دریافت شد. |
| **DC_EVENT_INCOMING_WEBXDC_NOTIFY** | یک webxdc می‌خواهد پیام اطلاعی یا خلاصه تغییر یافته اطلاع داده شود. |
| **DC_EVENT_INFO** | یک رشته اطلاعاتی در لاگ نوشته شد. |
| **DC_EVENT_LOCATION_CHANGED** | موقعیت مکانی یک یا چند مخاطب تغییر کرده. |
| **DC_EVENT_MSG_DELETED** | یک پیام تکی حذف شد. |
| **DC_EVENT_MSG_DELIVERED** | یک پیام تکی با موفقیت ارسال شد و توسط ارائه‌دهنده دریافت شد. |
| **DC_EVENT_MSG_FAILED** | یک پیام تکی ارسال نشد. |
| **DC_EVENT_MSG_READ** | یک پیام تکی توسط گیرنده به عنوان خوانده شده علامت‌گذاری شد. |
| **DC_EVENT_MSGS_CHANGED** | پیام‌ها یا چت‌ها تغییر کردند؛ اعلان عمومی برای بازنشانی رابط کاربری. |
| **DC_EVENT_MSGS_NOTICED** | پیام‌ها توسط کاربر به عنوان دیده شده علامت‌گذاری شدند. |
| **DC_EVENT_NEW_BLOB_FILE** | هنگامی منتشر می‌شود که یک فایل blob جدید (پیوست/آواتار) با موفقیت نوشته شد. |
| **DC_EVENT_OUTGOING_CALL_ACCEPTED** | تماس قرار داده شده با `dc_place_outgoing_call()` توسط گیرنده پذیرفته شد. |
| **DC_EVENT_REACTIONS_CHANGED** | واکنش‌های پیام (لایک، پاسخ‌های ایموجی) تغییر کرد. |
| **DC_EVENT_SECUREJOIN_INVITER_PROGRESS** | پیشرفت دست‌دهی پیوستن امن از دیدگاه دعوت‌کننده. |
| **DC_EVENT_SECUREJOIN_JOINER_PROGRESS** | پیشرفت دست‌دهی پیوستن امن از دیدگاه پیونده. |
| **DC_EVENT_SELFAVATAR_CHANGED** | آواتار خود کاربر تغییر کرد. |
| **DC_EVENT_SMTP_CONNECTED** | هنگامی منتشر می‌شود که اتصال SMTP برقرار شده و ورود موفقیت‌آمیز بوده. |
| **DC_EVENT_SMTP_MESSAGE_SENT** | هنگامی منتشر می‌شود که پیامی با موفقیت به سرور SMTP ارسال شد. |
| **DC_EVENT_TRANSPORTS_MODIFIED** | رله انتقال اضافه/حذف شده یا پیش‌فرض تغییر کرده. |
| **DC_EVENT_WARNING** | یک رشته هشدار در لاگ نوشته شد. |
| **DC_EVENT_WEBXDC_INSTANCE_DELETED** | پیامی که شامل نمونه webxdc بود حذف شد. |
| **DC_EVENT_WEBXDC_REALTIME_ADVERTISEMENT** | تبلیغ برای ارتباط کانال همتا به همتای موقت دریافت شد. |
| **DC_EVENT_WEBXDC_REALTIME_DATA** | داده از طریق کانال همتا به همتای موقت دریافت شد. |
| **DC_EVENT_WEBXDC_STATUS_UPDATE** | به‌روزرسانی وضعیت webxdc دریافت شد (برای همگام‌سازی وضعیت). |
