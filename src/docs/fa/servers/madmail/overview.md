# Madmail

**Madmail** یک نسخه تخصصی از [سرور ایمیل Maddy](https://maddy.email/) است که به‌طور خاص برای عمل به‌عنوان یک سرور **Chatmail** با کارایی بالا و راه‌اندازی آسان برای دلتاچت بهینه‌سازی شده است.

برخلاف رله استاندارد Chatmail (که از ترکیبی از Postfix، Dovecot و اسکریپت‌های پایتون استفاده می‌کند)، Madmail یک **راهکار تک‌فایلی (Single Binary)** ارائه می‌دهد که تمام اجزای لازم را در خود ادغام کرده است.

## ویژگی‌های کلیدی

- **تک‌فایلی**: بدون نیاز به مدیریت جداگانه نصب‌های Dovecot، Postfix و Nginx.
- **ثبت‌نام خودکار**: ایجاد حساب کاربری «در لحظه» (Just-in-time) در اولین ورود.
- **ادغام TURN**: پشتیبانی داخلی برای تماس‌های WebRTC.
- **پراکسی Shadowsocks**: پراکسی محدود فنی داخلی برای بهبود اتصال.
- **سیاست عدم ثبت لاگ (No Log Policy)**: مکانیزم‌های حریم خصوصی عمیقاً یکپارچه برای به حداقل رساندن ذخیره‌سازی متادیتا.
- **سیاست PGP-معتبر**: اجبار اختیاری رمزنگاری سرتاسری برای تمام ایمیل‌های ورودی و خروجی.

## چرا Madmail؟

Madmail برای کاربرانی طراحی شده است که یک سرور **جامع و آماده** می‌خواهند که نصب و نگهداری آن بسیار آسان باشد، به‌ویژه روی سخت‌افزارهایی با منابع محدود مانند Raspberry Pi یا سرورهای مجازی (VPS) کوچک.
