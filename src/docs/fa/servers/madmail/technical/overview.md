# مستندات فنی Madmail

این بخش شامل مستندات فنی عمیق درباره معماری Madmail، جزئیات پیاده‌سازی و مرجع ماژول‌ها است.

Madmail که به‌عنوان جایگزینی تک‌فایلی و با کارایی بالا برای پشته‌های سنتی ایمیل طراحی شده است، SMTP، IMAP و سرویس‌های کمکی مختلف (مانند TURN و Shadowsocks) را در یک سیستم منسجم ادغام می‌کند.

## بخش‌های کلیدی

- **[معماری داخلی](/fa/docs/servers/madmail/technical/quirks)**: درک تصمیمات طراحی و ذخیره‌سازی مبتنی بر SQLite.
- **[مرجع ماژول‌ها](/fa/docs/servers/madmail/technical/reference/modules-intro)**: پیکربندی دقیق تمام اجزای Madmail.
- **[آموزش‌ها](/fa/docs/servers/madmail/technical/tutorials/setting-up)**: راهنماهای فنی گام‌به‌گام.
- **[سیاست امنیتی](/fa/docs/servers/madmail/technical/no-log-policy)**: مستندات مربوط به سیاست عدم ثبت لاگ و اجبار PGP.
