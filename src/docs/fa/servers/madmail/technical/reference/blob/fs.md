# سیستم فایل (Filesystem)

این ماژول بدنه‌ی پیام‌ها را در یک دایرکتوری در سیستم فایل ذخیره می‌کند.

```hcl
storage.blob.fs {
    root <دایرکتوری>
}
```

```hcl
storage.blob.fs <دایرکتوری>
```

## دستورالعمل‌های پیکربندی

### `root [path]`
پیش‌فرض: تنظیم نشده است

مسیر دایرکتوری سیستم فایل. باید توسط فرآیند سرور قابل خواندن و نوشتن باشد. اگر وجود نداشته باشد، ایجاد خواهد شد (دایرکتوری والد باید برای این کار قابل نوشتن باشد). مسیرهای نسبی نسبت به دایرکتوری وضعیت (state directory) سرور تفسیر می‌شوند.
