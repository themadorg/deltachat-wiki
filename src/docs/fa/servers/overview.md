# سرورهای دلتاچت

دلتاچت با سرورهای استاندارد ایمیل کار می‌کند، اما ما نرم‌افزارهای سرور تخصصی را نیز ارائه می‌دهیم که به‌طور خاص برای عملکرد پیام‌رسانی، حریم خصوصی و سهولت استفاده طراحی شده‌اند.

## چرا یک سرور تخصصی؟

اگرچه دلتاچت می‌تواند از هر سرور ایمیلی استفاده کند، راهکارهای تخصصی مانند Chatmail و Madmail مزایای زیر را ارائه می‌دهند:

- **عملکرد پیام‌رسانی فوری** — بهینه‌سازی شده برای پیام‌های کوچک و مکرر.
- **حریم خصوصی** — بدون لاگ، بدون ردیابی و با حداقل ذخیره‌سازی متادیتا.
- **راه‌اندازی آسان** — سرور پیام‌رسان خود را در چند دقیقه راه‌اندازی کنید.
- **پیکربندی خودکار** — کلاینت‌های دلتاچت اغلب می‌توانند خود را به‌طور خودکار با این سرورها پیکربندی کنند.
