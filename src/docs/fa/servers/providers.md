# انتخاب سرویس‌دهنده ایمیل

در دلتاچت، «سرور» همان سرویس‌دهنده ایمیل شماست. شما می‌توانید از هر سرویس‌دهنده‌ای استفاده کنید، اما برخی برای چت کردن مناسب‌تر هستند.

## سرویس‌های پیشنهادی
- **Chatmail**: سرورهای بهینه‌سازی شده مخصوص دلتاچت که بسیار سریع هستند.
- **Posteo**: یک سرویس پولی با تمرکز بر حریم خصوصی.
- **Mailbox.org**: سرویس ایمیل آلمانی امن و پایدار.

  استفاده از سرورهای Chatmail تجربه چت را بسیار سریع‌تر (تقریباً آنی) می‌کند.
</DCFeatureCard>

## نکات مهم
برخی سرویس‌دهنده‌ها مانند Gmail یا Outlook ممکن است محدودیت‌هایی در تعداد پیام‌ها یا دسترسی IMAP داشته باشند. برای بهترین تجربه، استفاده از سرویس‌های مستقل توصیه می‌شود.
