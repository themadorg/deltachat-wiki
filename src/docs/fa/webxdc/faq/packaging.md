# بسته‌بندی

## بهینه‌سازی آیکون برنامه

چندین کار وجود دارد که می‌توانید برای کاهش اندازه آیکون خود انجام دهید:

- بدون تصویر بندانگشتی ذخیره کنید (در gimp می‌توان در دیالوگ صادرات انجام داد)
- رزولوشن تصویر را کاهش دهید (`256px` کافی است، در برخی موارد `128px` یا حتی کمتر مانند `64px` هم می‌تواند کافی باشد)
- رنگ‌های PNG خود را از `RGB` به `Indexed` تغییر دهید (در gimp: `Image` -> `Mode` -> `Indexed`، ببینید <https://docs.gimp.org/en/gimp-image-convert-indexed.html>)


> برای `png` همچنین می‌توانید از ابزار `oxipng` (<https://github.com/shssoichiro/oxipng>) استفاده کنید که به طور خودکار اندازه فایل آیکون شما را بدون از دست دادن کیفیت بهینه می‌کند:
> ```
> oxipng icon.png -s -o max
> ```
>
> اگر فایل‌های png در پروژه خود دارید، باید این کار را روی آن‌ها هم انجام دهید تا بایت‌های بیشتری صرفه‌جویی کنید.
>
> پارامترهای قابل توجه:
> - `--pretend` فقط سود را محاسبه می‌کند
> - `-Z` فشرده‌سازی بیشتر، اما بیشتر طول می‌کشد
> - برای اطلاعات بیشتر ببینید `oxipng --help`# عیب‌یابی
