# تاریخچه تغییرات

## [1.2] - 2024-12-15

### APIهای جدید

- تغییر `webxdc.selfAddr` برای برگرداندن شناسه هر-برنامه 
  که می‌تواند برای لیست «اعلان» جدید استفاده شود 

- افزودن ویژگی جدید `update.notify` برای ایجاد 
  اعلان سیستمی قابل مشاهده توسط کاربر تا آگاه شود
  که چیزی در برنامه تغییر کرده 

- افزودن ویژگی جدید `update.href` تا با کلیک روی پیام‌های اطلاعاتی
  «لینک عمیق» به داخل برنامه فراهم شود تا ناوبری آسان‌تر شود

- افزودن `webxdc.sendUpdateMaxInterval` و `webxdc.sendUpdateMaxSize` جدید
  محدودیت‌های نرخ که به برنامه‌ها کمک می‌کند تا با ویژگی‌ها/محدودیت‌های انتقال سازگار شوند


### منسوخ‌شده‌ها

- `webxdc.sendUpdate` دیگر آرگومان «descr» نمی‌گیرد. 


### تغییرات دیگر 

- انتقال ویژگی‌های «پشتیبانی‌شده» DOM/JavaScript از سؤالات متداول 
  به بخش مشخصات پیام‌رسان 

- بهبود نگارش در چندین مکان
