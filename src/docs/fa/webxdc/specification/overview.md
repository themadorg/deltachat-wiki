# مشخصات Webxdc

مشخصات webxdc سه حوزه را پوشش می‌دهد:

- [فرمت فایل کانتینر `.xdc`](./format) فرمت فایل zip
  با فایل‌های manifest و آیکون را توصیف می‌کند.

- [API جاوااسکریپت](./api) یک API حداقلی پایدار را مشخص می‌کند که تمام برنامه‌های webxdc
  می‌توانند استفاده کنند.

- [پیاده‌سازی پیام‌رسان](./messenger) نحوه اجرای برنامه‌های webxdc
  توسط پیام‌رسان در وب‌ویوهای ایزوله را مشخص می‌کند.
