# События ядра Delta Chat

Это полный список событий, генерируемых движком Delta Chat Core. Боты должны слушать эти события, чтобы реагировать на различные действия, такие как входящие сообщения, изменения в подключении или обновления конфигурации.

| Название события | Описание |
| :--- | :--- |
| **DC_EVENT_ACCOUNTS_BACKGROUND_FETCH_DONE** | Сообщает, что фоновое получение данных завершено (или истекло время ожидания). |
| **DC_EVENT_ACCOUNTS_CHANGED** | Сообщает, что список аккаунтов изменился (аккаунт удален или добавлен, или изменился порядок аккаунтов). Генерируется только менеджером аккаунтов. |
| **DC_EVENT_ACCOUNTS_ITEM_CHANGED** | Сообщает, что свойство аккаунта, такое как `is_configured`, `displayname`, `selfavatar` или `private_tag`, было изменено. |
| **DC_EVENT_CALL_ENDED** | Входящий или исходящий вызов завершен или не был принят в течение 1 минуты. |
| **DC_EVENT_CHANNEL_OVERFLOW** | Информирует, что некоторые события были пропущены из-за переполнения канала событий. |
| **DC_EVENT_CHAT_DELETED** | Чат был удален. |
| **DC_EVENT_CHAT_EPHEMERAL_TIMER_MODIFIED** | Изменен таймер исчезающих сообщений в чате. |
| **DC_EVENT_CHAT_MODIFIED** | Чат изменился (например, название, аватар или участники). |
| **DC_EVENT_CHATLIST_CHANGED** | Сообщает, что набор чатов или их порядок в списке чатов изменился. |
| **DC_EVENT_CHATLIST_ITEM_CHANGED** | Сообщает, что все или один элемент списка чатов изменился и требует перерисовки. Если `chat_id` равен 0, перерисовка требуется для всех видимых чатов. |
| **DC_EVENT_CONFIG_SYNCED** | Изменено значение конфигурации, синхронизируемое между несколькими устройствами. |
| **DC_EVENT_CONFIGURE_PROGRESS** | Информирует о прогрессе конфигурации, запущенной `dc_configure()`. |
| **DC_EVENT_CONNECTIVITY_CHANGED** | Изменилось состояние подключения к серверу (например, подключение, в сети, офлайн). |
| **DC_EVENT_CONTACTS_CHANGED** | Контакт(ы) создан, переименован, верифицирован, заблокирован или удален. |
| **DC_EVENT_DELETED_BLOB_FILE** | Генерируется при успешном удалении файла (изображения/вложения). |
| **DC_EVENT_ERROR** | Библиотека должна сообщить об ошибке конечному пользователю. |
| **DC_EVENT_ERROR_SELF_NOT_IN_GROUP** | Действие не может быть выполнено, так как требуется членство в группе. |
| **DC_EVENT_IMAP_CONNECTED** | Генерируется при установке соединения IMAP и успешном входе. |
| **DC_EVENT_IMAP_INBOX_IDLE** | Генерируется перед переходом в режим IDLE для папки Inbox. |
| **DC_EVENT_IMAP_MESSAGE_DELETED** | Генерируется, когда сообщение успешно помечено как удаленное на сервере IMAP. |
| **DC_EVENT_IMAP_MESSAGE_MOVED** | Генерируется при успешном перемещении сообщения по протоколу IMAP. |
| **DC_EVENT_IMEX_FILE_WRITTEN** | Файл был экспортирован (например, во время резервного копирования/экспорта). |
| **DC_EVENT_IMEX_PROGRESS** | Информирует о прогрессе импорта/экспорта, запущенного `dc_imex()`. |
| **DC_EVENT_INCOMING_CALL** | Получен входящий вызов. |
| **DC_EVENT_INCOMING_CALL_ACCEPTED** | Собеседник принял входящий вызов на этом или другом устройстве. |
| **DC_EVENT_INCOMING_MSG** | Получено новое сообщение, которое сохранено в базу данных. |
| **DC_EVENT_INCOMING_MSG_BUNCH** | Загрузка пачки сообщений только что завершена. |
| **DC_EVENT_INCOMING_REACTION** | Получена реакция на собственное отправленное сообщение. |
| **DC_EVENT_INCOMING_WEBXDC_NOTIFY** | Приложению webxdc требуется уведомить об информационном сообщении или изменении сводки. |
| **DC_EVENT_INFO** | Информационная строка записана в лог. |
| **DC_EVENT_LOCATION_CHANGED** | Местоположение одного или нескольких контактов изменилось. |
| **DC_EVENT_MSG_DELETED** | Одиночное сообщение удалено. |
| **DC_EVENT_MSG_DELIVERED** | Одиночное сообщение успешно отправлено и получено провайдером. |
| **DC_EVENT_MSG_FAILED** | Одиночное сообщение не удалось отправить. |
| **DC_EVENT_MSG_READ** | Одиночное сообщение помечено получателем как прочитанное. |
| **DC_EVENT_MSGS_CHANGED** | Сообщения или чаты изменились; общее уведомление для обновления пользовательского интерфейса. |
| **DC_EVENT_MSGS_NOTICED** | Сообщения были помечены пользователем как замеченные или просмотренные. |
| **DC_EVENT_NEW_BLOB_FILE** | Генерируется при успешной записи нового файла (вложения/аватара). |
| **DC_EVENT_OUTGOING_CALL_ACCEPTED** | Вызов, инициированный с помощью `dc_place_outgoing_call()`, был принят собеседником. |
| **DC_EVENT_REACTIONS_CHANGED** | Реакции на сообщение (лайки, ответы эмодзи) изменились. |
| **DC_EVENT_SECUREJOIN_INVITER_PROGRESS** | Прогресс рукопожатия Secure Join со стороны приглашающего. |
| **DC_EVENT_SECUREJOIN_JOINER_PROGRESS** | Прогресс рукопожатия Secure Join со стороны присоединяющегося. |
| **DC_EVENT_SELFAVATAR_CHANGED** | Собственный аватар пользователя изменился. |
| **DC_EVENT_SMTP_CONNECTED** | Генерируется при установке соединения SMTP и успешном входе. |
| **DC_EVENT_SMTP_MESSAGE_SENT** | Генерируется при успешной отправке сообщения на сервер SMTP. |
| **DC_EVENT_TRANSPORTS_MODIFIED** | Транспортный релей добавлен/удален, или изменился релей по умолчанию. |
| **DC_EVENT_WARNING** | Предупреждающая строка записана в лог. |
| **DC_EVENT_WEBXDC_INSTANCE_DELETED** | Удалено сообщение, содержавшее экземпляр webxdc. |
| **DC_EVENT_WEBXDC_REALTIME_ADVERTISEMENT** | Получено объявление об эфемерном канале связи peer-to-peer. |
| **DC_EVENT_WEBXDC_REALTIME_DATA** | Получены данные по эфемерному каналу peer-to-peer. |
| **DC_EVENT_WEBXDC_STATUS_UPDATE** | Получено обновление статуса webxdc (для синхронизации состояния). |
