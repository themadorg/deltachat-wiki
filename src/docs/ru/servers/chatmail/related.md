# Разработки сообщества

Активная разработка ведется в репозитории GitHub [chatmail/relay](https://github.com/chatmail/relay).

Вы можете изучить [тег 'chatmail' на форуме поддержки support.delta.chat](https://support.delta.chat/tag/chatmail) и попросить добавить вас в непубличный чат поддержки для отладки проблем.

Нам известно о трех проектах альтернативных реализаций, находящихся в разработке:

- **[Mox](https://github.com/mjl-/mox)**: почтовый сервер на Golang. Ведется работа по поддержке всех функций и настроек, необходимых для работы в качестве реле chatmail.
- **[Madmail](https://github.com/omidz4t/madmail)**: экспериментальный форк почтового сервера [Maddy](https://maddy.email/), оптимизированный для развертывания в качестве chatmail. Предоставляет решение в виде одного бинарного файла.
- **[Chatmail Cookbook](https://github.com/feld/chatmail-cookbook)**: Chef Cookbook для реализации сервера-реле. Поддерживает FreeBSD и использует DNS-01 для валидации сертификатов.
