# Madmail

**Madmail** — это специализированный форк [почтового сервера Maddy](https://maddy.email/), оптимизированный для работы в качестве высокопроизводительного и простого в развертывании сервера **Chatmail** для Delta Chat.

В отличие от стандартного реле Chatmail (которое использует комбинацию Postfix, Dovecot и сценариев на Python), Madmail представляет собой **решение в виде одного бинарного файла**, объединяющее все необходимые компоненты.

## Ключевые особенности

- **Один бинарный файл**: нет необходимости управлять отдельными установками Dovecot, Postfix и Nginx.
- **Автоматическая регистрация**: создание учетных записей «на лету» (Just-In-Time) при первом входе в систему.
- **Интегрированный TURN**: встроенная поддержка видеозвонков WebRTC.
- **Прокси Shadowsocks**: интегрированный ограниченный прокси для улучшения качества связи.
- **Политика отсутствия логов**: глубоко интегрированные механизмы обеспечения приватности для минимизации хранения метаданных.
- **Политика PGP-Only**: опциональное принудительное использование сквозного шифрования для всей входящей и исходящей почты.

## Почему именно Madmail?

Madmail предназначен для пользователей, которым нужен сервер формата **«всё включено»**, который чрезвычайно легко устанавливать и обслуживать, особенно на оборудовании с ограниченными ресурсами, таком как Raspberry Pi или небольшие VPS-инстансы.
