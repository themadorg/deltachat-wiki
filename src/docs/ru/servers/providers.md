# Выбор почтового провайдера

В Delta Chat «сервером» является ваш почтовый провайдер. Вы можете использовать любого провайдера, но некоторые лучше подходят для общения в чате.

## Рекомендуемые сервисы
- **Chatmail**: специализированные серверы для Delta Chat, которые работают очень быстро.
- **Posteo**: платный сервис с акцентом на приватность.
- **Mailbox.org**: надежная и стабильная немецкая почтовая служба.

## Важные советы
Некоторые провайдеры, такие как Gmail или Outlook, могут иметь ограничения на количество сообщений или доступ по протоколу IMAP. Для лучшего опыта рекомендуется использовать независимые сервисы.
