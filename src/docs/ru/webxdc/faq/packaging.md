# Упаковка

## Оптимизация иконки приложения

Есть несколько способов уменьшить размер иконки вашего приложения:

-   Сохраняйте без миниатюры (thumbnail) — в GIMP это можно сделать в диалоге экспорта.
-   Уменьшите разрешение изображения (достаточно **256px**, в некоторых случаях хватит **128px** или даже **64px**).
-   Измените цветовой режим PNG с `RGB` на `Indexed` (в GIMP: `Изображение` -> `Режим` -> `Индексированный`, подробнее в [документации GIMP](https://docs.gimp.org/en/gimp-image-convert-indexed.html)).

> Для оптимизации `png` вы также можете использовать инструмент `oxipng` ([репозиторий на GitHub](https://github.com/shssoichiro/oxipng)), который автоматически оптимизирует размер файла без потери качества:
> ```
> oxipng icon.png -s -o max
> ```
> 
> Если в вашем проекте много PNG-файлов, прогоните их через этот инструмент, чтобы сэкономить ещё больше байтов.
>
> Полезные параметры:
> - `--pretend` — только рассчитывает экономию без изменения файлов.
> - `-Z` — еще более сильное сжатие, но занимает больше времени.
> - Для получения дополнительной информации введите `oxipng --help`.
