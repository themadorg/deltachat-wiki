# Список изменений

## [1.2] — 15.12.2024

### Новые API

-   Изменено поведение `webxdc.selfAddr`: теперь возвращается идентификатор для конкретного приложения, который можно использовать в списке уведомлений `notify`.
-   Добавлен атрибут `update.notify`, позволяющий вызывать системные уведомления для привлечения внимания пользователя к изменениям в приложении.
-   Добавлен атрибут `update.href`, чтобы нажатие на информационные сообщения (`info`) позволяло переходить по прямым ссылкам внутри приложения.
-   Добавлены лимиты `webxdc.sendUpdateMaxInterval` и `webxdc.sendUpdateMaxSize`, помогающие приложениям адаптироваться к характеристикам транспорта.

### Устаревшие функции (Deprecations)

-   Функция `webxdc.sendUpdate` больше не принимает аргумент `descr`.

### Прочие изменения

-   Раздел «Поддерживаемые функции DOM/Javascript» перенесен из FAQ в спецификацию мессенджера.
-   Улучшены формулировки во многих частях спецификации.
