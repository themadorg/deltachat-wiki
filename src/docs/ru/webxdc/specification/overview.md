# Спецификация Webxdc

Спецификация webxdc охватывает три области:

-   [Формат файла контейнера `.xdc`](format.md) — описывает формат zip-файла с манифестом и иконкой.

-   [Javascript API](api.md) — определяет минимальный стабильный API, который могут использовать все приложения webxdc.

-   [Реализация в мессенджере](messenger.md) — определяет, как мессенджеры запускают приложения webxdc в изолированных веб-представлениях (webviews).
