# Send Webxdc app from composer attachment menu

Composer **📎** → **Webxdc app** (`webxdc_app`) opens `AppPicker` overlay. Pick app → download `.xdc` → `addFileToDraft` → send as Webxdc message. Hidden for outbound broadcast channels.

**Verify:** attach → Webxdc app; pick from store; send; Webxdc bubble appears in chat.
