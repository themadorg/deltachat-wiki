# Recent Webxdc icons in chat navbar (up to 3)

`ChatView` → `AppIcons` from `getChatMedia(..., 'Webxdc')`, most recently updated first. Up to **3** icons ( **1** on small-screen). Click opens app.

**Verify:** use webxdc apps in chat; navbar shows up to 3 recent app icons; click opens app.
